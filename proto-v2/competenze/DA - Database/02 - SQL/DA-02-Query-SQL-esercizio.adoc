==== Scenario

Stai lavorando a un sistema di e-commerce.

Il Product Owner richiede un report con i prodotti più venduti negli ultimi 30 giorni.

==== Materiale fornito

[source,text]
----
Customer
--------
Id
Name

CustomerOrder
-------------
Id
CustomerId
OrderDate

OrderItem
---------
Id
OrderId
ProductId
Quantity
UnitPrice

Product
-------
Id
Name
Category
----

==== Consegna

Scrivi una query SQL che restituisca, per ogni prodotto venduto negli ultimi 30 giorni:

* nome del prodotto;
* categoria;
* quantità totale venduta;
* ricavo totale generato.

Il risultato deve essere ordinato dal prodotto con il ricavo maggiore a quello con il ricavo minore.
